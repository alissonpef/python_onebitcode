## Módulos da Formação

Este repositório está organizado de acordo com os módulos da formação:

- 🐍 **Python I - Introdução ao Python:** Fundamentos, tipos de dados (strings, listas, tuplas, sets, dicionários), operadores, laços de repetição e funções.
- 🧩 **Python II – Módulos e PIP:** Criação/uso de módulos, ambientes virtuais e gerenciamento com PIP.
- 🏗️ **Python III - POO:** Classes, objetos, encapsulamento, herança, polimorfismo e decorators.
- 📂 **Python IV - Manipulando Arquivos:** Leitura e escrita (TXT, CSV), manipulação de dados e PDFs.
- 📊 **Banco de Dados com SQL:** Fundamentos, modelagem de dados e consultas SQL.
- 💾 **Python V - Integração com Banco de Dados:** Interação com SQLite, PostgreSQL e MongoDB.
- 🌿 **Git e GitHub:** Controle de versão e colaboração.

## Como Iniciar

Siga as instruções abaixo para executar os códigos deste repositório localmente.

### Pré-requisitos

Certifique-se de ter o Python instalado na sua máquina.
* Python
  Faça o download no site oficial: [python.org](https://www.python.org/downloads/)

### Instalação

1. Clone o repositório
   ```sh
   git clone https://github.com/alissonpef/python_onebitcode.git
   ```
2. Navegue até o diretório desejado e execute o script alvo
   ```sh
   python nome_do_arquivo.py
   ```
